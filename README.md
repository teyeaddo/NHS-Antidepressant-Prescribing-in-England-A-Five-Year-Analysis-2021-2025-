# NHS-Antidepressant-Prescribing-in-England-A-Five-Year-Analysis-2021-2025-
This repo contains files and data submitted as an assignment from a data analysis training class.
Research Question: How has antidepressant prescribing in England changed between 2021 and 2025 — in terms of volume, cost, drug type, and regional variation — and what do these patterns mean for NHS planning and mental health policy?
